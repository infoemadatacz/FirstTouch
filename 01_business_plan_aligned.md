It applies **Touch It Once** to enterprise operations: intake once, classify once, route once, execute with control.

Critical positioning rule:
- Do not position First Touch as a standalone side project.
- Position it as an **operational layer inside the EMA AI system direction**.

## 5. Commercial Model

### Offer ladder

1. Diagnostic Sprint (2 weeks)
   - Workflow map, loss baseline, pilot shortlist, ROI model.
2. Constrained Pilot (6-8 weeks)
   - One high-value workflow, human-in-the-loop, before/after metrics.
3. Scale Program (quarterly)
   - Connector depth, policy expansion, governance hardening.

### Pricing logic

Hybrid:
- Fixed platform/program fee.
- Variable fee linked to governed workflow execution.
- Optional outcome-linked upside for mature deployments.

## 6. ROI Frame

Primary value drivers:
- Executive attention reclaimed.
- Reduced manual routing and re-entry.
- Lower exception handling overhead.
- Faster intake-to-resolution cycle.

Baseline formula:

`Annual Value = (Hourly Value x Weekly Hours Saved x 50) - Annual System Cost`

Sales rule:
- Always present low/base/high assumptions.
- Show payback in months, not abstract percentages.

## 8. Governance and Risk

Main risks:
- Prompt injection and unsafe action triggering.
- Cross-team permission leaks.
- Over-automation beyond acceptable risk boundaries.

Governance design:
- System/user instruction separation.
- Least-privilege connector permissions.
- Explicit human checkpoint thresholds.
- Exportable action logs for audit and compliance.
