# First Touch
## Sales Ops and CRM System

Last update: 2026-02-24

## 1. Pipeline Stages

1. Targeted
2. Connected
3. Warming
4. Qualified
5. Call Invited
6. Call Booked
7. Call Held
8. Diagnostic Proposed
9. Diagnostic Won
10. Pilot Proposed
11. Pilot Won
12. Closed Lost

## 2. Stage Exit Criteria

### Targeted -> Connected
- connection accepted

### Connected -> Warming
- prospect replied at least once

### Warming -> Qualified
- pain confirmed + owner hinted

### Qualified -> Call Invited
- relevance confirmed in current quarter

### Call Invited -> Call Booked
- date/time confirmed

### Call Held -> Diagnostic Proposed
- workflow and scope discussed

### Diagnostic Proposed -> Diagnostic Won
- commercial agreement signed

## 3. CRM Field Requirements

Required fields:
- company
- persona
- industry
- tier
- source
- current stage
- next action
- owner

Recommended fields:
- pain summary
- workflow candidate
- objections
- call notes
- close probability

## 4. Weekly KPI Dashboard

Core metrics:
- acceptance rate
- reply rate
- qualification rate
- invite-to-book rate
- show rate
- diagnostic close rate

Secondary metrics:
- time in stage
- no-response rate
- lost reason distribution

## 5. Weekly Review Ritual (30-45 min)

1. Stage conversion review
2. Lost reason review
3. Message performance review
4. ICP quality review
5. Next-week experiments

## 6. Data Hygiene Rules

- Update pipeline daily.
- Every record must have a next action date.
- No stage moves without evidence.
- Close stale threads explicitly.

## 7. Performance Thresholds

- Acceptable acceptance rate: >= 30%
- Acceptable DM reply rate: >= 20%
- Acceptable show rate: >= 70%
- Acceptable diagnostic close rate: >= 30%

If below threshold for 2 weeks:
- tighten ICP
- adjust opening lines
- rework CTA phrasing

## 8. Ownership Model

- Jan owns high-value conversations and calls.
- Assistant workflow owns tracking, reminders, sequence pacing.
