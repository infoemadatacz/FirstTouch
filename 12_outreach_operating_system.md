# First Touch
## Outreach Operating System

Last update: 2026-02-24

## 1. Daily Production Targets

Per working day:
- 15-25 quality connection requests
- 10 high-value comments on target accounts
- 10-20 follow-up DMs
- 1-2 call invitations (only qualified threads)

## 2. Weekly Targets

- 75-125 connection requests
- 50 meaningful comments
- 60-100 DM follow-ups
- 6-12 qualified call invitations
- 3-8 calls booked

## 3. Outreach Workflow

### Step 1: Build Account List
Tag accounts by:
- ICP tier (A/B/C)
- persona (CEO/COO/RevOps/CS/IT)
- industry
- current relevance trigger

### Step 2: Warm First
Before request:
- react to one post
- leave one real comment when possible

### Step 3: Send Connection Request
Use scripts from `04_message_library_and_timing.md`.

### Step 4: Start EMA-led Conversation
- no pitch
- no call ask
- one relevance angle

### Step 5: Qualify
Use one diagnostic question.

### Step 6: Offer Working Session
Only after intent signal.

## 4. Message Cadence Rules

- Keep messages under 5 lines.
- One CTA per message.
- No attachments before request.
- No call invitation in first DM.

## 5. Content + Outreach Sync

After each published post:
1. collect engagers from target personas
2. send 15-25 targeted requests
3. use post angle as DM context

## 6. ICP Prioritization Rules

Prioritize threads where prospect confirms:
- active bottleneck this quarter
- clear operational owner
- willingness to run constrained pilot

Deprioritize threads with:
- generic curiosity only
- no owner
- no urgency

## 7. Weekly Review Questions

- Which hook generated best reply quality?
- Which persona converts to calls fastest?
- Where is First Touch introduced too early?
- Which objections repeatedly block movement?

## 8. Escalation Rules

Escalate copy/messaging update when:
- acceptance rate drops for 2 consecutive weeks
- DM response drops below baseline
- call show rate drops below 60%

## 9. Anti-Burnout Execution Rule

Operate in focused blocks:
- 45 min account research
- 45 min comments + requests
- 45 min follow-ups
- 30 min pipeline updates
