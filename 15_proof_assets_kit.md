# First Touch
## Proof and Trust Assets Kit

Last update: 2026-02-24

## 1. Required Assets

1. Workflow bottleneck map template
2. Diagnostic sprint one-pager
3. Pilot scorecard template
4. Case study template
5. Governance threshold worksheet

## 2. Diagnostic One-Pager Structure

- Client context (anonymized)
- Workflow in scope
- Current pain metrics
- Proposed intervention
- Expected KPI movement
- Scope boundaries
- Commercial summary

## 3. Pilot Scorecard Template

Track weekly:
- cycle time
- rework rate
- escalation rate
- executive intervention count
- exception quality notes

## 4. Case Study Template

Sections:
- client profile
- baseline challenge
- pilot scope
- implementation steps
- measured outcomes
- governance notes
- scale decision

## 5. Trust Signals to Include on LP and in DMs

- operator context in regulated workflow environments
- governance-first approach
- constrained pilot discipline
- measurable outcome framing

## 6. What Not to Use as "Proof"

- vanity impressions
- model benchmark screenshots
- generic claims without workflow metrics

## 7. Delivery Format

- LP snippets for fast trust
- PDF one-pager for post-call send
- short written case note for DM follow-up
